🎯 吸顶头部功能

这个演示展示了插件中的吸顶头部效果。当您滚动页面时，顶部的文件信息栏和导航按钮会始终保持在顶部可见。

  • ✅ 文件名、类型、大小信息始终可见
  • ✅ 上一个/下一个按钮随时可用
  • ✅ 半透明背景和模糊效果
  • ✅ 平滑的过渡动画
  • ✅ 响应式设计

🖼️ 图片预览区域

这里是图片预览的主要区域。当图片很大时，用户可以滚动查看，而文件信息和导航按钮始终保持在顶部。

图片预览区域

📱 响应式支持

吸顶头部在不同屏幕尺寸下都能正常工作：

  • 桌面端：完整显示所有信息
  • 平板端：自适应调整间距
  • 手机端：紧凑布局，保持可用性

🌙 暗色主题适配

吸顶头部支持暗色主题，会根据系统主题自动调整：

  • 浅色主题：白色半透明背景
  • 暗色主题：深色半透明背景
  • 平滑的主题切换动画

⚡ 性能优化

吸顶效果经过优化，确保流畅的用户体验：

  • 使用 CSS sticky 定位，性能更好
  • backdrop-filter 硬件加速
  • 节流的滚动事件处理
  • 平滑的过渡动画

🎨 视觉效果

吸顶头部具有现代化的视觉效果：

  • 半透明背景增加层次感
  • 模糊背景突出内容
  • 阴影效果增强立体感
  • 滚动时的动态反馈
